### BINARY 겨울방학 웹 프로젝트

> ✏️ 주제 : 웹페이지 플래시 게임 구현 <br>
> 🗓️ 기간 : 23.01.02 ~ 23.02.27 <br>
> 🛠️ 개발 언어 : `html`, `css`, `javascript` <br>

<br>

### 🔥 팀 소개

---

- **팀명** : **WeBinary**
    
    `We + Binary` or `Web + Binary` → 중의적인 표현
    
- **팀원**
    - **전경원 :** 팀장 / 점수 계산 및 페이지 새로고침 기능 구현
    - **김원준 :** 웹페이지 디자인 및 프레임 구현
    - **김민세 :** 골키퍼 및 공 움직임 기능, 웹페이지 이미지 표시 구현

<br>

### 🎮 게임 설명

---

⚽ 승부차기 게임 - AI mode

<br>

🏆  **승리 조건**

승부차기 **5라운드** 진행 후 **3점 이상** 득점

❓ **득점 시** (골이 들어갔을 경우)

화면에 GOAL 이미지가 뜨고 SCORE +1점

❓ **실패 시** (골을 넣지 못할 경우)

화면에 MISS 이미지가 뜨고 SCORE 유지

<br>

🕹️  **조작 방법**

- 방향(SIDE) 조절 : ↔️ `좌우 방향키`를 통해 `-3(왼쪽) ~ 3(오른쪽)` 까지 조절 가능
- 세기(POWER) 조절 : ↕️ `상하 방향키`를 통해 `0~3` 까지 조절 가능
- 슈팅 : 스페이스 바

<br>

🖥️  게임 시연

- `webinary.html` 을 누르면 게임 접속이 가능합니다.
1. 메인화면에서 **Kick it!** 버튼을 누르면 인게임 화면으로 넘어간다.

<img stype="margin:50px 0 10px 0" src="https://user-images.githubusercontent.com/96944509/222110960-a24f019e-74b6-4317-913a-2c06292cc98a.png" width="50%" height="50%" />

2. 인게임 화면 우측에 있는 **SIDE**, POWER 는 각각 **방향**, **세기**를 나타낸다.
   SIDE는 **±**3까지 조작이 가능하고 POWER는 0 ~ 3까지 조작이 가능하다. 
    
<img stype="margin:50px 0 10px 0" src="https://user-images.githubusercontent.com/96944509/222111543-254c568d-5eea-49df-a41d-45996b58233f.png" width="50%" height="50%" />    

3. 먼저 좌우 방향키로 방향을 조절하고, 상하 방향키로 세기를 조절한 뒤 스페이스바를 입력하면 슈팅이 나간다.

- 골이 들어가면 화면에 GOAL 이미지가 뜨고 SCORE가 올라간다.

- 골을 넣지 못할 경우에는 화면에 MISS 이미지가 뜨고 SCORE가 올라가지 않는다.
 
4. 5라운드 진행 후 게임이 종료된다.

- 3점 이상 득점하였을 경우 승리

- 3점 미만으로 득점하게 되면 패배

5. 게임이 종료되면 화면 중앙 하단에 있는 Restart 버튼을 누르게 되면 다시 게임이 시작된다.
